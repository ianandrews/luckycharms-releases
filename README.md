# LuckyCharms Releases

Public release downloads for LuckyCharms on macOS.

## Download

1. Open the latest release in this repo.
2. Download the `.dmg` file.
3. Open the `.dmg` and drag `LuckyCharms.app` into `Applications`.

## First Launch

1. Open `Applications/LuckyCharms.app`.
2. If macOS warns that the app was downloaded from the internet, click `Open`.
3. Follow the setup flow in the app.
4. Grant Microphone and Screen Recording permissions when prompted.
5. Add your Anthropic API key if you want AI meeting notes.

## Notes

- Apple Silicon is required.
- macOS 14 or later is required.
- The first recording downloads the local transcription model, so that first start can take a little longer.
- Auto-updates use the assets in this repository.
